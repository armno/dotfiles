# Armno's dotfiles

> note to self: `sync.sh` is not used.

### vim

```sh
$ cd ~/code
$ hub clone armno/dotfiles
$ ln -s ~/code/dotfiles/.vimrc ~/.vimrc
$ ln -s ~/code/dotfiles/.vim ~/.vim
```

### git

```sh
$ ln -s ~/code/dotfiles/.gitconfig ~/.gitconfig
```

### osx
```sh
$ ln -s ~/code/dotfiles/.osx ~/.osx
```
